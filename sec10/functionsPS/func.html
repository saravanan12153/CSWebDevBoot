Functions Problem Set

  1. Write a function isEven() which takes a single numeric agrument and returns true if the number is even, and false otherwise.
  2. Write a function factorial() which takes a single numeric argument and returns the factorial of that number. factorial(4) is 4 x 3 x 2 x 1. 0 is 1.
  3. Write a function kebabToSnake() which takes a single kebab-cased string argument and returns the snake_cased version. kebabToSnake("hello-world"); is "hello_world"
